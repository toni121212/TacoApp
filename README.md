# TacoApp — Sistema de Pedidos para Taquería
TacoApp permite que los clientes realicen pedidos desde la mesa mediante un código QR, que los meseros gestionen los pedidos desde su celular y que la cocina reciba la información en tiempo real. Además, busca reducir errores en los pedidos, agilizar el servicio y facilitar el control de ventas para el dueño o administrador.
Framework: Flutter  |  Backend: Firebase
Metodología: Scrum (sprints de 2 semanas)
